# SQL Server AdventureWorks Scripts

## 📌 Overview
This repository contains SQL Server scripts and Power BI dashboards demonstrating:
- **Indexes**: clustered and non-clustered indexing strategies.
- **Schema**: database schema design and relationships.
- **Queries**: practical SQL scripts (triggers, procedures, joins, CTEs, filtering, CRUD operations).
- **Dashboards**: Power BI visualizations with screenshots and related queries.

The purpose of this repository is to showcase **end-to-end BI workflow skills**:
- Writing efficient SQL queries.
- Designing and indexing tables for performance.
- Building interactive dashboards in Power BI.
- Documenting results with clear structure and screenshots.
