# 📂 Queries - SQL Scripts Portfolio

## Overview
This folder contains a collection of **SQL scripts** that demonstrate practical skills in:
- Data querying and filtering  
- Aggregation and reporting  
- Joins, subqueries, and CTEs  
- Constraints, triggers, and stored procedures  
- Indexing and performance optimization  

Each script highlights a specific SQL concept or use case, making this folder a clear portfolio of hands-on database skills.

---

## 📑 Scripts Included
- **Address_Audit_Trigger.sql** → Implements a trigger to track changes in the Address table.  
- **Assign_RowNumber_Persons_ByCity.sql** → Uses window functions to assign row numbers per city.  
- **Calculate_Net_Sales_With_CTE.sql** → Calculates net sales per staff using a Common Table Expression (CTE).  
- **CRUD_Operations.sql** → Demonstrates Create, Read, Update, and Delete operations.  
- **Customer_Filter_Subquery_Exists.sql** → Filters customers using a correlated subquery with `EXISTS`.  
- **Data_Filtering_And_Sorting.sql** → Shows practical filtering and sorting techniques.  
- **HighValue_Customers_Report_GroupBy.sql** → Generates a report of high-value customers using `GROUP BY`.  
- **IndexedView_Product_Description.sql** → Creates an indexed view for product descriptions.  
- **OrderPriority_Classification_CASE_2011.sql** → Classifies orders using `CASE` expressions.  
- **Product_Filtering_Procedure.sql** → Implements a stored procedure for product filtering.  
- **Product_SalesOrder_LeftJoin.sql** → Demonstrates `LEFT JOIN` for product sales orders.  
- **Yearly_ShipMethod_Freight_PIVOT.sql** → Uses `PIVOT` to analyze yearly freight costs by shipping method.
