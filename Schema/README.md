# Schema Scripts

هذا المجلد يحتوي على سكربتات SQL الخاصة بإنشاء الجداول والقيود في قاعدة بيانات **AdventureWorks**.

## الملفات
- **01_tables.sql** → إنشاء الجداول الأساسية.  
- **02_constraints.sql** → إضافة القيود (Primary Key, Foreign Key, Unique, Check).  

## الهدف
توضيح مهاراتي في تصميم قواعد البيانات باستخدام SQL Server من خلال بناء هيكل منظم للجداول والقيود، كجزء من بورتفوليو عملي.

## طريقة الاستخدام
1. افتح SQL Server Management Studio.  
2. نفّذ السكربتات بالترتيب (الجداول أولًا ثم القيود).  
3. تأكد من نجاح التنفيذ قبل الانتقال إلى الاستعلامات أو الفهارس.
